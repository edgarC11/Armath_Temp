Go over project 22 from the elegoo guide on the lcd display.

LCD display wiring:

VSS -> GND
VDD - > 5V
VE -> potentiometer
RS -> D7
RW - > GND
E -> D8
D0 - D3 -> Skip
D4 -> D9
D5 -> D10
D6 -> D11
D7 -> D12
BA -> 5V
BC -> GND

thermistor resistor: 10k Ohm
thermistor pin: A0

LED resistors: 220Ohm

LED pins: 
coldPin = 22;
greenPin = 24;
yellowPin = 26;
redPin = 28;
